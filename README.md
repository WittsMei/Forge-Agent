# Forge-Agent


### What it can do?
- Create a game, such as Snake Game, and modify the code simply one sentence.



### Agent Operating Rules
- Originality and IP Safety Rules

The agent must create original software, games, characters, names, mechanics, visual designs, and assets.

The agent must not copy, imitate, or recreate copyrighted games, brands, characters, sprites, sounds, maps, names, or protected assets.

For example, the agent must not create a “Super Mario” game, use Nintendo names, use Mario-like characters, copy Mario-style sprites, or reproduce recognizable franchise elements.

If the user asks for a copyrighted clone, the agent should transform the request into an original project inspired only by the broad genre.
